Installation

  1. Make sure you have BetterDiscord installed
  2. Download the theme files from the GitHub repository
  3. Place Darker.theme.css in your BetterDiscord themes folder:
    • Windows: %appdata%\Roaming\BetterDiscord\themes\
    • Mac: ~/Library/Application Support/BetterDiscord/themes/
    • Linux: ~/.config/BetterDiscord/themes/
  4. Place the src folder in the same location
  5. Enable the theme in BetterDiscord settings

Customization

Darker offers extensive customization options through CSS variables. Open the Darker.theme.css file in any text editor to modify these settings:

Color Scheme

--darker-bg-1: #101014; /* Darkest background (main chat area) */
--darker-bg-2: #16161c; /* Second level background (sidebars) */
--darker-bg-3: #1c1c24; /* Third level background (server list, popups) */
--darker-bg-4: #232330; /* Fourth level background (cards, modals) */
--darker-accent-h: 260; /* Accent color hue (0-360) */
--darker-accent-s: 80%; /* Accent color saturation */
--darker-accent-l: 65%; /* Accent color lightness */

Layout & Styling

--darker-radius-ui: 6px; /* UI elements radius */
--darker-radius-avatar: 20%; /* Avatar radius (in percent, 50% is circle) */
--darker-radius-server: 20%; /* Server icon radius */
--darker-spacing-app: 10px; /* App outer margins */
--darker-spacing-ui: 8px; /* General UI spacing */

Extra Features

To enable any of the extra features, add one or more of these classes to :root in your Darker.theme.css file:

:root {
--add: darker-compact-mode; /* Makes everything smaller and more compact */
--add: darker-centered-mode; /* Centers the chat for better readability */
--add: darker-rounded-mode; /* Makes UI elements more rounded */
--add: darker-glass-mode; /* Adds transparency and blur effects */
--add: darker-custom-status-bars; /* Enhances status indicators */
--add: darker-amoled-mode; /* Pure black backgrounds for AMOLED displays */
